AI Job API Documentation

Detailed technical documentation, workflows, and API specifications.

1. System Overview

The AI Job API is a FastAPI backend designed to:

  • Aggregate Jobs: Scrape listings from Indeed, LinkedIn, ZipRecruiter, and Google using jobspy.
  • Generate Content: Use Google Gemini AI to create professional resume summaries and descriptions.
  • Cache Data: Utilize MongoDB to store search results for performance optimization.

2. Workflows

A. Job Search Workflow

1. Request: User sends search_term and location.
2. Cache Check: System queries MongoDB (jobs_cache).
  • Hit: Returns cached data immediately. Triggers background refresh.
  • Miss: Calls fetch_and_cache.
3. Scraping: jobspy scrapes multiple platforms (Indeed, LinkedIn, etc.).
4. Storage: Results are cleaned and stored in MongoDB (TTL: 2 days).

B. AI Generation Workflow

1. Request: User sends JSON body (e.g., skills, experience).
2. Prompting: System constructs a prompt for Google Gemini based on input.
3. Generation: Calls Gemini API to generate text.
4. Response: Returns parsed JSON suggestions.

3. API Specifications

GET /api/jobs

Fetch job listings from multiple sources.

Query Parameters

Parameter Type Required Description
search_term string Yes Job title or keyword (e.g., "Python Developer")
location string Yes City or region (e.g., "New York")
country_indeed string No Country code (default: "India")

GET /api/internships

Fetch internship listings.

Query Parameters

Parameter Type Required Description
title string Yes Internship title (e.g., "Data Science")
location string Yes City or region

POST /api/summary

Generate a professional profile summary based on skills.

Request Body (JSON)

{
  "skills": ["python", "fastapi", "react"],
  "summary": "Optional existing summary if rewriting",
  "isRewrite": false
}

Response

{
  "data": ["Summary Option 1...", "Summary Option 2..."],
  "message": true
}

POST /api/internship-summary

Generate bullet points for an internship experience.

Request Body (JSON)

{
  "designation": "Software Intern",
  "company": "Tech Corp",
  "summary": "Optional existing summary",
  "isrewrite": false
}

POST /api/experience-summary

Generate bullet points for professional work experience.

Request Body (JSON)

{
  "designation": "Senior Developer",
  "company": "Big Tech Inc",
  "summary": "Optional existing summary",
  "isrewrite": false
}

POST /api/accomplishment-summary

Polish or generate an accomplishment description.

Request Body (JSON)

{
  "title": "Won National Hackathon",
  "description": "Optional details",
  "isrewrite": false
}
